🧠 MesoNet Architecture

~28K params
Input 256×256×3
↓
Conv2D(8, 3×3) + BN + Pool Block 1
↓
Conv2D(8, 5×5) + BN + Pool Block 2
↓
Conv2D(16, 5×5) + BN + Pool Block 3
↓
Conv2D(16, 5×5) + BN + Pool(4) Block 4
↓
Flatten → Dense(16) → Dense(1) Sigmoid output
Reference: Afchar, D. et al. (2018). MesoNet: a Compact Facial Video Forgery Detection Network. IEEE WIFS.
